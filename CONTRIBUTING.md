# Contributing

Grazie per il tuo interesse nel contribuire a questo progetto!

## Setup del progetto

1. Clona il repository
2. Installa le dipendenze: `npm install`
3. Copia `.env.example` in `.env.local` e configura le variabili
4. Avvia il server di sviluppo: `npm run dev`

## Sviluppo

- Usa `npm run dev` per lo sviluppo
- Usa `npm run build` per testare la build di produzione
- Usa `npm run lint` per controllare il codice
- Usa `npm run format` per formattare il codice

## Pull Requests

1. Crea un branch per la tua feature
2. Fai le tue modifiche
3. Testa che tutto funzioni
4. Crea una pull request con una descrizione chiara
